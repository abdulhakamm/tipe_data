Tipe data

tipe data adalah sebuah pengklasifikasian data berdasarkan jenis data

Tipe Data Number

Tipe data yang mendukung tipe data number isa berupa bilanagan bulat maupun desimal

Tipe Data Boolean

Tipe data yang berisikan data kebenaran,tipe data boolean terdiri dari dua data yaitu true dan false

Variable

Variable adalah tempat untuk menyimpan data, dengan variable kita bisa menggunakan data tersebut tanpa mengubah secara manual
